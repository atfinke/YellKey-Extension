# YellKey Browser Extensions
A browser extension for quick [yellkey](http://www.yellkey.com) generation and entry.

## Fork Diffs

- Animations
- Multi-Window Support
- YellKey Entry (respects ↵ vs ⌘↵)
- Delta Orange Spinner
- Higher Res Icon
